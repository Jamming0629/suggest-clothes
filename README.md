# AI ファッションアドバイザー

楽天ファッションから服を検索して、ユーザーの好みに合わせた服を提案するWebアプリケーションです。

## 機能

- スタイル、色、シーン、季節、体型、身長に基づく服の検索
- AIによる服の提案と楽天ファッションでの検索
- 価格、ブランド、商品画像の表示
- 楽天での購入ページへの直接リンク

## セットアップ

### 1. 依存関係のインストール

```bash
npm install
```

### 2. 環境変数の設定

1. プロジェクトのルートディレクトリに`.env.local`ファイルを作成
2. 以下の内容を追加：

```bash
# OpenAI API設定
# OpenAIのウェブサイト（https://platform.openai.com/）からAPIキーを取得してください
OPENAI_API_KEY=sk-your-openai-api-key-here
```

### 3. OpenAI APIキーの取得

1. [OpenAI Platform](https://platform.openai.com/)にアクセス
2. アカウントを作成またはログイン
3. 「API Keys」セクションで新しいAPIキーを作成
4. 作成されたAPIキーを`.env.local`ファイルに設定

### 4. アプリケーションの起動

```bash
npm run dev
```

## 使用方法

1. アプリケーションを起動
2. 好みの設定（スタイル、色、シーン、季節、体型、身長）を選択
3. 「AIで服をサジェストする」ボタンをクリック
4. AIからの提案を受け取り、楽天ファッションで検索
5. 検索結果からお気に入りの服を見つけて、楽天で購入

## 技術スタック

- **フロントエンド**: Next.js 14, React, TypeScript
- **スタイリング**: Tailwind CSS
- **AI**: OpenAI GPT-4 API
- **画像生成**: OpenAI DALL-E 3 API
- **検索**: 楽天ファッション

## AI機能について

このアプリケーションはOpenAIの最新技術を使用しています：

- **GPT-4**: 服の提案とスタイリングアドバイス
- **DALL-E 3**: 服の画像生成
- **楽天ファッション連携**: 実際の商品検索と購入

## 注意事項

- OpenAI APIキーは`.env.local`ファイルに設定され、Gitにコミットされません
- APIキーは外部には送信されません

- 楽天ファッションの利用規約に従ってご利用ください
